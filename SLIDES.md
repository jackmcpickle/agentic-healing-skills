# Slides — editable export

Round-trippable export of every slide in `src/pages/index.astro` (in order).
Edit the values below, hand this file back, and the `.astro` will be regenerated to match.

## How to edit

- **One YAML block per slide.** Keep `slide:` order; reorder blocks to reorder slides.
- **`{…}` = green highlight** (`text-primary`). e.g. `Flow {state}` → "state" is green. Works in `heading`, `lead`, `quote`.
- **Only include the fields a slide uses.** Drop a field to remove that element.
- **`type`** picks the layout: `title · content · menu · image · mermaid · demo · quote · close · thanks`.
- **`section`**: name string puts the slide inside that section (and in the footer). `null`/omit = intro or close.
- **`icon`**: component name from `src/components/icons/` (e.g. `Waves`, `Compass`). Omit = no icon.
- **`draft: true`** shows the DRAFT badge.
- **`lead`** = primary subheading. **`muted`** = secondary, dimmer subheading.
- **`quote` + `cite`** = blockquote with attribution. **`eyebrow`** = small label above content. **`subtext`** = footer line (speaker/credits).
- **`custom`**: free-text for generated/special components (Menu spine, Mermaid flows). Describes what renders + how to change it — edit the referenced source, not raw markup here.
